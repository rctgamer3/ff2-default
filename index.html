Firefox 2, the theme, reloaded by rctgamer3, fax8 and borr

Full power of the new Gecko 2 engine with a good old Firefox 2 face.
Home Screenshots Bugs & Issues

The default Firefox 2 theme for Firefox 4 (and up) with some customizations, for (even) better results use the oldbar and Status-4-Evar extensions.
The theme is perfectly suited for all of the Firefox 4+ users which still think that Firefox 2 looked better and don't want to get used to the new look.
